# microlisp

[![MIT License](http://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

Small LISP Interpreter.

## Documentation

This is a Lisp interpreter, which is so small, easy to read and understand.

### BUILD

```console
$ make
```

### RUN

```console
$ ./bin/lisp <file>
```

### TEST

```console
$ make test
```

### CLEAN

```console
$ make clean
```
